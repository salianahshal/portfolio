DocFlow

DocFlow simplifies task completion by enabling users to upload all necessary documents for a task and seamlessly switch between them, eliminating the need to navigate through various tabs.

Background

This website was created to streamline document management, especially for tasks involving multiple documents. As my first React project, it provided a practical way to learn this JavaScript library while meeting a real need. The goal was to build an intuitive platform that simplifies document uploading and navigation, boosting user productivity and efficiency.

Role

User research, human-centered design, visual design.

Time

5 weeks, Feb. - April, 2024

Visual Design

Design Principles

  • Simplicity and Clarity
    • Clean, minimalistic design with a focus on functionality.
    • Clear labels and intuitive icons to enhance usability.
  • Responsiveness
    • Ensures the UI is fully responsive and accessible on various devices, including desktops, tablets, and smartphones.

Home Page

The Home Page provides a simple and intuitive interface for users to upload and manage their documents. It features an upload button, and a list of uploaded documents with options to view.
